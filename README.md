# Smart Behavioral Tracking Engine (SBTE)

## Overview
The Smart Behavioral Tracking Engine (SBTE) is an AI-powered behavioral intelligence platform designed for fintech applications. It tracks user interaction patterns, analyzes behavioral signals, predicts intent, and generates personalized interventions to improve engagement and conversion.

This project was built for a 6-day hackathon, focusing on providing actionable insights beyond traditional analytics.

## Core Features
- **Behavioral Event Tracking:** Monitors clicks, page visits, dwell time, and interaction patterns.
- **Intelligent Decision Engine:** Evaluates user behavior against a hybrid system of rule-based logic and lightweight Machine Learning.
- **Personalized Recommendations:** Triggers automated messages (e.g., email, in-app notifications) based on detected intent (Hesitation, High Intent, Confusion).
- **User Segmentation:** Groups users into personas (Beginner, Experienced, Premium) using clustering.
- **Admin Insights:** Provides a dashboard for monitoring user flows and engine performance.

## Tech Stack
- **Engine:** Python (Pandas, Scikit-learn, NumPy)
- **Backend API:** FastAPI (intended)
- **Database:** PostgreSQL / MongoDB (managed by teammates)
- **Frontend:** React + Tailwind CSS (managed by teammates)

## Project Structure
- `engine/`: Contains the core behavioral analysis and recommendation logic.
- `LOGIC.md`: Detailed explanation of the behavioral heuristics and ML models.
- `RUN.md`: Instructions on how to set up and execute the engine.
- `GEMINI.md`: Project requirements and hackathon strategy.
